# Introduction

Herald is built on Next.js with a [Supabase](https://supabase.io) backend. Our docs cover deployment to [Vercel](/docs/deployment/vercel) and [Netlify](/docs/deployment/netlify) can be hosted anywhere that can run Node.js.
